## Como usar este portal

- Use o **Diário das Sessões** para lembrar o que aconteceu.
- Use **Personagens do Grupo** para consultar quem é quem.
- Use **Pistas e Handouts** quando quiser rever documentos e descobertas.
- Use **Mundo Conhecido** para revisar lugares e nomes importantes.

> [!warning] Regra do portal
> Se uma informação não está aqui, ela ainda não é conhecimento público da campanha.
