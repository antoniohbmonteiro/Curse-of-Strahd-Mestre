# Ideia — Baróvia reconhece os personagens

## Premissa

Baróvia não conhece os personagens, mas reconhece padrões: culpa, fome, orgulho, dívida, violência, perda, desejo de proteger.

## Aplicação rápida

- A casa mostra para Lionel uma cena de acusação injusta.
- Para Svetlana, luxo apodrecido por exploração.
- Para Dhorak, uma porta que parece grade de cela.
- Para Kael, sangue que chama investigação.
- Para Oryn, um caminho que se dobra como portal.
- Para Yann, um batimento dentro da parede.

## Cuidado

Não revelar tudo. Só cutucar.
